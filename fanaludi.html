Nuestra Empresa y Propósito

Somos FaNaLuDi Software, un equipo dedicado a la ingeniería de software, comprometido con la creación de soluciones innovadoras y de alta calidad. Nuestro nombre, FaNaLuDi, fue elegido por todos los integrantes, simbolizando las primeras dos letras de cada uno: Fabrizio, Nahuel, Lucas y Diego. Esta fusión busca ser fácil de recordar y pronunciar, reflejando nuestra unión como equipo.

El logo de la empresa, también elegido por el equipo, está compuesto por las iniciales de cada integrante siguiendo el orden del nombre de la empresa. Formado por cuatro iniciales sutilmente fusionadas y la palabra "software" debajo, en un color negro, el logo es minimalista, limpio y elegante, fácil de adaptar a diferentes contextos.

Proyecto S.I.G.P.D. - Draftosaurus

Nuestro proyecto actual es el Sistema Interactivo de Gestión de Partidas de Draftosaurus (S.I.G.P.D.), una aplicación web diseñada para mejorar la experiencia del popular juego de mesa Draftosaurus. A continuación, detallamos los aspectos clave de nuestro modelo de negocio y desarrollo para este proyecto.

Funcionalidades Clave:

  • Modo de Seguimiento: Puntuación automática para partidas físicas.
  • Modo de Juego Digital: Replicación completa de las mecánicas del juego original.
  • Cálculo de Puntajes: Automatizado según las reglas del juego.
  • Validación de Movimientos: Asegurando que los jugadores sigan las reglas.
  • Historial y Estadísticas: Guarda y muestra el historial de partidas y estadísticas por jugador.
  • Multijugador Local: Permite partidas para 2 a 5 jugadores en la misma máquina o red.

Tecnologías Utilizadas:

Desarrollado con tecnologías libres como PHP 8.x, MariaDB, HTML, CSS, JavaScript y Bootstrap. El sistema está diseñado para ser ligero y no requiere conexión a internet externa.
